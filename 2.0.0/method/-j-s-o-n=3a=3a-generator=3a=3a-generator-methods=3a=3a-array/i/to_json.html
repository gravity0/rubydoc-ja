instance method JSON::Generator::GeneratorMethods::Array#to_json

to_json(state = nil) -> String

自身から生成した JSON 形式の文字列を返します。

[PARAM] state:
生成する JSON 形式の文字列をカスタマイズするために JSON::State のインスタンスを指定します。
