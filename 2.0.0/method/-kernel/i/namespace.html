instance method Kernel#namespace

namespace(name = nil) { ... } -> Rake::NameSpace [added by rake]

新しい名前空間を作成します。

与えられたブロックを評価する間は、その名前空間を使用します。

例: 

ns = namespace "nested" do
  task :run
end
task_run = ns[:run] # find :run in the given namespace.
