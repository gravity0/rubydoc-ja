instance method Gem::Version#bump

bump -> Gem::Version

最後の一桁を切り上げた新しい Gem::Version のインスタンスを返します。

例: 

ver1 = Gem::Version.create('5.3.1') # => #<Gem::Version "5.3.1">
ver2 = ver1.bump                    # => #<Gem::Version "5.4">
