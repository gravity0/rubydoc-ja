instance method Addrinfo#unix?

unix? -> bool

アドレスが Unix domain socket のものならば true を返します。 

Addrinfo.tcp("127.0.0.1", 80).unix? #=> false
Addrinfo.tcp("::1", 80).unix?       #=> false
Addrinfo.unix("/tmp/sock").unix?    #=> true
