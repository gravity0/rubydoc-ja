instance method Gem::Server#quick

quick(request, response) -> ()

メソッド名に対応する URI に対するリクエストを処理するメソッドです。

[PARAM] request:
WEBrick::HTTPRequest オブジェクトが自動的に指定されます。
[PARAM] response:
WEBrick::HTTPResponse オブジェクトが自動的に指定されます。
