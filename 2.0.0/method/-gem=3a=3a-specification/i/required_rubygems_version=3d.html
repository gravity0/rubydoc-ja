instance method Gem::Specification#required_rubygems_version=

required_rubygems_version=(requirement)

この Gem パッケージを動作させるのに必要な RubyGems のバージョンをセットします。

[PARAM] requirement:
Gem::Requirement.create が受け付ける形式のオブジェクトを指定します。
