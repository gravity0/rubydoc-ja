singleton method IRB::INSPECTORS.keys_with_inspector

keys_with_inspector(inspector) -> Array

引数で指定した IRB::Inspector に対応する key の配列を返します。

[PARAM] inspector:
IRB::Inspector オブジェクトを指定します。 
IRB::INSPECTORS.keys_with_inspector(IRB::INSPECTORS[true])
# => [true, :p, "p", :inspect, "inspect"]
