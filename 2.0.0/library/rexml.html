library rexml

要約

Pure Ruby の XML パーサです。 DOM スタイルと SAX スタイルの両方をカバーしています。

  • DOMっぽいAPI
    • rexml/document
    • rexml/element
    • rexml/attribute
    • rexml/namespace
    • rexml/text (おまけで rexml/cdata も？)
    • rexml/instruction
  • SAXっぽいAPI
    • rexml/parsers/streamparser
    • rexml/streamlistener
  • rexml/parsers/sax2parser
  • rexml/sax2listener
