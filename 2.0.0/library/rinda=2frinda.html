library rinda/rinda

要約

Rubyで実装されたタプル空間(Tuple Space)を扱うためのライブラリです。

  • http://www.druby.org/ilikeruby/rinda.html
  • http://www2a.biglobe.ne.jp/~seki/ruby/d208.html
