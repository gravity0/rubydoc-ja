class FalseClass

クラスの継承リスト: FalseClass < Object < Kernel < BasicObject

要約

false のクラス。 false は FalseClass クラスの唯一のインスタンスです。 false は nil オブジェクトとともに偽を表し、 その他の全てのオブジェクトは真です。
