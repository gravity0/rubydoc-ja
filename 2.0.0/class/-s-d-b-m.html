class SDBM

クラスの継承リスト: SDBM < Enumerable < Object < Kernel < BasicObject

要約

SDBM ファイルをアクセスするクラス。

キー、データともに文字列でなければならないという制限と、 データがファイルに保存されるという点を除いては Hash クラスと 全く同様に扱うことができます。

@see Hash
