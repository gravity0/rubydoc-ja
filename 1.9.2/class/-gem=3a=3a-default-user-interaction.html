module Gem::DefaultUserInteraction

クラスの継承リスト: Gem::DefaultUserInteraction

要約

このモジュールはデフォルトの Gem::UserInteraction を定義しています。

特異メソッド

定義 説明
ui -> Gem::ConsoleUI

デフォルトの UI を返します。

ui=(new_ui)

デフォルトの UI を新しくセットします。

use_ui(new_ui) { ... }

与えられたブロックを評価している間だけ UI として new_ui を使用します。
