module JSON::Generator::GeneratorMethods::String

クラスの継承リスト: JSON::Generator::GeneratorMethods::String
aliases: JSON::Ext::Generator::GeneratorMethods::String

要約

String に JSON で使用するインスタンスメソッドを追加するためのモジュールです。

インスタンスメソッド

定義 説明
to_json -> String

自身から生成した JSON 形式の文字列を返します。

to_json_raw -> String

自身に対して String#to_json_raw_object を呼び出して Hash#to_json した結果を返します。

to_json_raw_object -> Hash

生の文字列を格納したハッシュを生成します。
