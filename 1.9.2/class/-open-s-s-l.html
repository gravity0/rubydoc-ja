module OpenSSL

クラスの継承リスト: OpenSSL

要約

OpenSSL のすべてのクラス、モジュール、メソッド、定数を 保持しているモジュールです。

モジュール関数

定義 説明
debug -> bool

デバッグモードが on ならば true を返します。

debug=(b)

デバッグモードを on/off します。

errors -> [String]

OpenSSL のエラーキューに残っているエラー文字列を返します。

定数

定義 説明
OPENSSL_VERSION -> String

システムにインストールされている OpenSSL 本体のバージョンを表した文字列です。

OPENSSL_VERSION_NUMBER -> Integer

システムにインストールされている OpenSSL 本体のバージョンを表した数です。 http://www.openssl.org/docs/crypto/OPENSSL_VERSION_NUMBER.html も参照してください。

VERSION -> String

Ruby/OpenSSL のバージョンです。
