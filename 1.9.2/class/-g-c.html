module GC

クラスの継承リスト: GC

要約

GC は Ruby インタプリタの「ゴミ集め(Garbage Collection)」を制御 するモジュールです。

@see ObjectSpace

特異メソッド

定義 説明
disable -> bool

ガーベージコレクトを禁止します。

enable -> bool

ガーベージコレクトを許可します。

start -> nil

ガーベージコレクトを開始します。

stress -> bool
stress=(value)

GC.stress が真に設定されている間は、GC を行えるすべての機会に GC を行います。

インスタンスメソッド

定義 説明
garbage_collect -> nil

ガーベージコレクトを開始します。
