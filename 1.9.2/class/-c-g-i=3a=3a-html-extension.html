module CGI::HtmlExtension

クラスの継承リスト: CGI::HtmlExtension

要約

HTML を生成するためのメソッドを提供するモジュールです。

例: 

cgi.a("http://www.example.com") { "Example" }
  # => "<A HREF=\"http://www.example.com\">Example</A>"
