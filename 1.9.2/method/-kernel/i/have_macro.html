instance method Kernel#have_macro

have_macro(macro, headers = nil, opt = "") -> bool [added by mkmf]
have_macro(macro, headers = nil, opt = "") { ... } -> bool [added by mkmf]

与えられた macro が共通のヘッダファイルか headers に定義されている場合は真を返します。 そうでない場合は偽を返します。

[PARAM] macro:
マクロの名前を指定します。
[PARAM] headers:
追加のヘッダファイルを指定します。
[PARAM] opt:
C コンパイラに渡すコマンドライン引数を指定します。
