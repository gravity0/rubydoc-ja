instance method Gem::GemPathSearcher#matching_files

matching_files -> [String]

与えられた spec に path が含まれている場合、その path のリストを返します。

[PARAM] spec:
Gem::Specification のインスタンスを指定します。
[PARAM] path:
探索対象のパスを指定します。
