module function FileTest.#size

size(file) -> Integer

ファイルのサイズを返します。

[PARAM] file:
ファイル名を表す文字列か IO オブジェクトを指定します。
[EXCEPTION] Errno::EXXX:
filename が存在しない場合、あるいはシステムコールに失敗した場合に発生します。
[EXCEPTION] IOError:
指定された IO オブジェクト file が既に close されていた場合に発生します。
