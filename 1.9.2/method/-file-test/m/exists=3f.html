module function FileTest.#exists?

exists?(file) -> bool

このメソッドはobsoleteです。

ファイルが存在する時に真を返します。 FileTest.#exist?を使って下さい。

[PARAM] file:
ファイル名を表す文字列か IO オブジェクトを指定します。
[EXCEPTION] IOError:
指定された IO オブジェクト file が既に close されていた場合に発生します。
