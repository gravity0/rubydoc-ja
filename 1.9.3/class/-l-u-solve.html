module LUSolve

クラスの継承リスト: LUSolve

要約

LU 分解を用いて、連立1次方程式 Ax = b の解 x を求めるモジュールです。

モジュール関数

定義 説明
ludecomp(a, n, zero = 0, one = 1) -> [Integer]

n 次正方行列を表す配列 a を破壊的に変更し、返り値と併せて元の行列の LU 分解を提供します。

lusolve(a, b, ps, zero = 0.0) -> [BigDecimal]

LU 分解を用いて、連立1次方程式 Ax = b の解 x を求めて返します。
