class YAML::DBM

クラスの継承リスト: YAML::DBM < DBM < Enumerable < Object < Kernel < BasicObject

要約

DBM の値に文字列以外も格納できるように拡張したクラスです。

インスタンスメソッド

定義 説明
self[key] -> object | nil

データベースからキーを探して対応する要素の値を返します。

self[key] = value
store(key, value) -> object

key に対して value を格納します。

delete(key) -> object | nil

key をキーとする要素を削除します。

delete_if {|key, val| ... } -> YAML::DBM

ブロックを評価した値が真であれば該当する要素を削除します。

each -> YAML::DBM
each_pair {|key, val| ... } -> YAML::DBM

自身のキーと値を引数としてブロックを評価します。

each_value {|val| ... } -> YAML::DBM

値を引数としてブロックを評価します。

fetch(key, ifnone = nil) -> object

データベースからキーを探して対応する要素の値を返します。

has_value?(value) -> bool

value を値とする組がデータベース中に存在する時、真を返します。

index(value) -> String | nil

value を持つ要素のキーを返します。

invert -> {object => String}

値からキーへのハッシュを返します。

reject {|key, val| ... } -> {String => object}

ブロックを評価した値が真であれば該当する要素を削除します。

replace(other) -> YAML::DBM

self の内容を other の内容で置き換えます。

select(*keys) -> [object]
select { ... } -> [[String, object]]

ブロックを評価して真になった要素のみを [キー, 値] から構成される配列に 格納して返します。ブロックが与えられなかった場合は、keys に対応する値を 配列に格納して返します。

shift -> [String, object]

データベース中のキー、値を一つ取り出し、データベースから削除します。

to_a -> [[String, object]]

キーと値のペアを配列に変換して返します。

to_hash -> Hash

自身のキー、値をハッシュにしたものを返します。

update(other) -> self

自身と other の内容をマージ(統合)します。

values -> object

データベース中に存在する値全てを含む配列を返します。

values_at(*keys) -> [object]

keys に対応する値を配列に格納して返します。

定義 説明
