class MatchData

クラスの継承リスト: MatchData < Object < Kernel < BasicObject

要約

正規表現のマッチに関する情報を扱うためのクラス。

このクラスのインスタンスは、

  • Regexp.last_match
  • Regexp#match, String#match
  • $~

などにより得られます。

インスタンスメソッド

定義 説明

n 番目の部分文字列を返します。

Range オブジェクト range の範囲にある要素からなる部分配列を返します。
