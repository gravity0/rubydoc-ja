class Psych::Nodes::Scalar

クラスの継承リスト: Psych::Nodes::Scalar < Psych::Nodes::Node < Enumerable < Object < Kernel < BasicObject

要約

YAML の scalar http://yaml.org/spec/1.1/#id858081 を表すクラスです。

これは AST の葉にあたるノードであり、子ノードを持ちません。

定義 説明
