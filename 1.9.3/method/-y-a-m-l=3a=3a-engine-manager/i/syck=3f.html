instance method YAML::EngineManager#syck?

syck? -> bool

syck ライブラリを使用中の場合に true を返します。それ以外の場合 は false を返します。 

require "psych"
require "yaml"
p YAML::ENGINE.syck? # => false

YAML::ENGINE.yamler = "syck"
p YAML::ENGINE.syck? # => true
