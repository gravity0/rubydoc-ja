module function FileUtils.#mkdir

mkdir(dir, options = {}) -> ()

ディレクトリ dir を作成します。

[PARAM] dir:
作成するディレクトリ。
[PARAM] options:
:mode, :noop, :verbose が指定できます。 FileUtils/オプションの説明

例: 

FileUtils.mkdir('test')
FileUtils.mkdir(%w( tmp data ))
FileUtils.mkdir('notexist', {:noop => true})  # does not create really
