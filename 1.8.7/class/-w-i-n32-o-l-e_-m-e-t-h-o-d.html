class WIN32OLE_METHOD

クラスの継承リスト: WIN32OLE_METHOD < Object < Kernel

要約

OLEオートメーションサーバが持つメソッドの情報を提供します。

WIN32OLE_METHODは、WIN32OLE#ole_methodsなどの呼び出しによって返さ れるオブジェクトで、OLEオートメーションサーバのメソッドの情報（メタデー タ）を保持します。

サンプルコード

  excel = WIN32OLE.new('Excel.Application')
  excel.ole_methods.each do |method|
    if method.visible?
      puts <<SIGNATURE
#{method.return_type} #{method.name}(#{
    method.params.map {|p| "#{p.ole_type} #{p.name}"}.join(', ')
}) : #{method.helpstring}
SIGNATURE
    end
  end
