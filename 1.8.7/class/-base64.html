module Base64

クラスの継承リスト: Base64

要約

Base64 エンコード / デコードするためのメソッドを定義したモジュールです。

モジュール関数

定義 説明
b64encode(bin, len = 60) -> ()

文字列 bin をエンコードし、len の長さで折り返し、表示します。

decode64(str) -> String

Base64エンコードされた文字列strをデコードします。

decode_b(str) -> String

[RFC2047] で定義されている encoded-word を含む 文字列 str をデコードします。

encode64(str) -> String

文字列strをBase64エンコードします。
