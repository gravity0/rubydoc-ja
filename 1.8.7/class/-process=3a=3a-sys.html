module Process::Sys

クラスの継承リスト: Process::Sys

要約

ユーザ ID・グループ ID を操作するシステムコールを直接呼ぶためのモジュールです。

ポータブルにユーザ ID・グループ ID を操作するためのモジュール Process::UID, Process::GID も提供されています。Process::Sys と Process::UID や Process::GID を同時に使うことは 非推奨です。
