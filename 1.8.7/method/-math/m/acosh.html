module function Math.#acosh

acosh(x) -> Float

x の逆双曲線余弦関数の値を返します。

定義

acosh(x) = log(x + sqrt(x * x - 1)) [x >= 1]
[PARAM] x:
x >= 1 の範囲の実数
[EXCEPTION] Errno::EDOM:
範囲外の値を引数に指定した場合に発生します。
acosh(z) -> Complex [redefined by complex]

複素関数としてのacosh関数(双曲逆cos関数)の結果を返します。

acosh関数は以下で定義されます。 

acosh(z) = log(z + sqrt(z*z-1))
[PARAM] z:
複素数
