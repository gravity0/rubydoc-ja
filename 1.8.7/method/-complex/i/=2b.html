instance method Complex#+

self + c -> Complex

複素数 c を加えた結果を返します。

[PARAM] c:
加算する数
[RETURN]
加算結果をComplexクラスのオブジェクトとして返します。

例: 

c =  Complex(1, 1)    #=> Complex(1, 1)
p c + Complex(3, 3)   #=> Complex(4, 4)
p c + 3               #=> Complex(4, 1)
