instance method RUNIT::Assert#assert_match

assert_match(str, re, message = "") -> ()
assert_matches(str, re, message = "") -> ()

与えられた文字列が与えられた正規表現にマッチする場合、検査にパスします。

[PARAM] str:
文字列を指定します。
[PARAM] re:
正規表現を指定します。
[PARAM] message:
検査に失敗したときのメッセージを指定します。
