instance method IRB::ExtendCommandBundle#install_alias_method

install_alias_method(to, from, override = NO_OVERRIDE)

メソッドのエイリアスを定義します。ライブラリ内部で使用します。

[PARAM] to:
新しいメソッド名を Symbol か文字列で指定します。
[PARAM] from:
元のメソッド名を Symbol か文字列で指定します。
[PARAM] override:
新しいメソッド名が定義済みであった場合にそのメソッドを 上書きするかどうかを IRB::ExtendCommandBundle::NO_OVERRIDE、 IRB::ExtendCommandBundle::OVERRIDE_PRIVATE_ONLY、 IRB::ExtendCommandBundle::OVERRIDE_ALL のいずれか で指定します。
