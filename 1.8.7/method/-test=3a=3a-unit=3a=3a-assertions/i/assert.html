instance method Test::Unit::Assertions#assert

assert(boolean, message = nil) -> ()

boolean が真ならパスします。

[PARAM] boolean:
検証するオブジェクトを指定します。
[PARAM] message:
assert が失敗した時に表示するメッセージを文字列で指定し ます。指定しなかった場合は表示しません。
[EXCEPTION] Test::Unit::AssertionFailedError:
assert が失敗した時に発生します。
