library csv

要約

CSV (Comma Separated Values) を扱うライブラリです。

クラスとモジュール

class CSV

CSV (Comma Separated Values) を扱うクラスです。

class CSV::BasicWriter

CSVデータ書き込み時に用いられるクラス

class CSV::Cell

CSV形式データのカラム（Cell）を操作するためのクラス。 このクラスはバージョン1.8.7で廃止されました。

class CSV::IOBuf

IOのバッファリングを実現するクラス

class CSV::IOReader

CSVの読み込み処理を行うクラス

class CSV::Reader

CSVの読み込み時のベースとなるクラス

class CSV::Row

CSV形式1行分のデータ（Row）を操作するためのクラス。

class CSV::StreamBuf

ストリームバッファを扱うクラス。

class CSV::StringReader

CSV形式の文字列の読み込み（パース）時に使用されるクラス

class CSV::Writer

CSVの書き込み時のベースとなるクラス

例外クラス

class CSV::IllegalFormatError

CSV 形式のパースが失敗した場合に発生します。
